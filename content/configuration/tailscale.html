Tailscale

The Tailscale plugin allows remote access to the Unraid server by joining to a predefined private zero-trust network zero-trust mesh network, where every device connects directly to each other via an encrypted WireGuard-based tunnel.

The server is connected to the Tailscale network (aka "Tailnet") accessed with my Github account. Configuration of the plugin is minimal and is done with Tailscale pages. The plugin seems to merely connect you to the correct webpage at Tailscale.com.

Before updating the plugin schedule a reboot ~30 minutes ahead using the at utility. This should allow you to reconnect to the server if the plugin fails to reconnect to Tailscale.
