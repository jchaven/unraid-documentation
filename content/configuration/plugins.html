Plug-ins

Plugins extended the functionality of Unraid by adding services that are not made by the Unraid developers and assist with maintenance. Most plugins are made by the Unraid community to satisfy personal needs and generously shares their efforts with the Unraid community.

Some features added to Unraid started out as plugins that were so useful they were incorporated into Unraid by the developers.

Appdata Backup

Unraid stores all files for the Docker Containers in a share named Appdata. If something were to happen to a container that prevented it from starting (like a failed update) then the container can be deleted and recreated using a backup copy of the contents of Appdata.

Community Applications

The Community Applications plugin, aka CA, provides an App Store-like experience where you can find other plugins, Docker containers and easily install them on an Unraid server.

Dynamix System Temperature

Dynamix System Temperature shows in real-time the temperature of the system CPU and motherboard. Temperatures can be displayed in Celsius or Fahrenheit.

Fix Common Problems

This must-have plugin runs several checks in the Unraid configuration and devices for errors that should be corrected.

NerdTools

This plugin replaces "NerdPack" which can add several utilities for use in the CLI. This plugin is not really needed but, does allow installing the tmux utility which can be useful for long-running commands during setup.

Tailscale

The Tailscale plugin is the client that connects the Unraid server the Tailnet. This is required for remote access to the server.

Unassigned Devices and Unassigned Devices Plus

Unassigned Devices, aka UD, adds the ability to mount disks in Unraid outside of the array. This is needed to connect and share USB external harddrives.

Unraid Connect

Unraid Connect is an online service for server management, monitoring, and maintenance. This feature of Unraid can provide remote access to Unraid servers but, requires port forwarding on the internet router. If the router is behind a CGNAT router (owned by ISP) then this feature will not work.

User Scripts

This plugin allows running scripts without having to manually alter the GO script.
