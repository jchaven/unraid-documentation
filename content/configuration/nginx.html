NGINX Web Server

Nginx, pronounced like "engine-ex", is an open-source web server that serves this website to clients locally and on the internet. The maintainers of this container (Linux Server IO) have included the latest version of PHP as well.

Container Properties

The container selected on the Apps tab in Unraid is the package maintained by the repository: lscr.io/linuxserver/nginx.

File Location

The HTML/PHP files served by the web server are stored in the "appdata" share and accessible as a "user" share. The following image is the location of the root directory as seen in a Linux terminal.

The same location as viewed in file manager on a Linux client.
