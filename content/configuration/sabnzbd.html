SABnzbd

The Docker container SABnzbd is the service that downloads the files requested by Sonarr for TV shows available in Kodi.

Container Properties

The container selected on the Apps tab in Unraid is the package maintained by "Linuxserver.io".

The configuration for the container is simple, you just need to enter the locations for the Complete and Incomplete downloads section. Also select an unused port on the Unraid server to which you connect using a browser.
