Cloudflare

The Tunnel daemon creates an encrypted tunnel between your origin web server and Cloudflare's nearest data center, all without opening any public inbound ports.

The use of tunneling is required because the network the server in High Point is behind a router provided by Lumos (North State) that is configured for Carrier-grade NAT (CGNAT). This means that forwarding port traffic to specific devices on the local network is not possible. This container is not installed on the the server in Charlotte.

The configuration for the container is simpler than most. Essentially, you just need to enter the token for the Cloudflare tunnel into the "Post Arguments" section.

Cloudflare properties

The tunnel created with Cloudflare is part of their "Zero Trust" applications. Log into Cloudflare and go to Zero Trust > Networks > Tunnels

The credentials to log into Cloudflare are stored in KeePass and require the two-factor authenticator app 2FAS on a cell phone.
