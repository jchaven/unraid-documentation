Sonarr

The Sonarr Docker container is the service that manages the TV shows available in Kodi. It monitors TV shows watching for content that meets the requirements defined for each show.

Container Properties

The container selected on the Apps tab in Unraid is the package maintained by "Linuxserver.io".

On the configuration page enter the locations for the for the "TV" pointing to the "Media" share and point to the "Downloads" share for the "Downloads" setting. To write content to the other media share "Media2" you'll need to add another path - it this case named "TV2". Also select an unused port on the Unraid server to which you connect using a browser.
