Virtual Machines

Unraid is a hypervisor that can emulate multiple virtual "computers" that are isolated from each other allowing virtual machines to run Linux and Windows operating systems at the same time.

Virtual Machines are not required to use Unraid and none currently configured on either server is critical.

Configured VMs

The following table lists all the VMs on the server in High Point (there are none in Charlotte).

Name Description
Ubuntu 5.0 Generic Ubuntu Server instance used to make clones of other VMs.
Ubuntu 5.1 Ubuntu Server instance used to download content not managed in Sonarr or manually ripped from discs.
Windows 10 A Windows 10 Pro instance used to do Windows things.
