Docker Containers

A Docker container image is a lightweight, standalone, executable package of software that includes everything needed to run an application: code, runtime, system tools, system libraries and settings.

Using Docker containers the Unraid server is able host web-based applications for various purposes. In the case of the Unraid server in High Point those applications are mainly ones that manage the content for the Kodi media shares.

Cloudflare

The Cloudflare container is the end-point for a Zero Trust (Argo) tunnel provided by Cloudflare. This allows access from the Internet to this website running in the NGINX container by routing traffic through the tunnel. This means open ports and port-forwarding in the router is not needed.

NGINX Web Server

This container runs the NGINX web server which hosts this website.

SABnzbd and Sonarr

These containers are used to manage the content for Kodi

Syncthing

Files on the two servers are kept in sync by running Syncthing in this container.
