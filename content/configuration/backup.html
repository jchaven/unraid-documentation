Backups

There currently is no automated backup process in place. Performing a backup is a manual process and requires the Unraid Plugin Unassigned Devices be installed.

The backups are done using a script attached to specific USB devices. The script is defined by clicking the "gear" icon for the mount of the USB device on the UD section on the Main screen in Unraid.

JBOD Backup

Files are backed-up to a Mediasonic HFR2-SU3S2 PRORAID 4 Bay 3.5" SATA Hard Drive Enclosure via a USB connection to the server. The Proraid box is configured as a JBOD (Just a Bunch Of Discs) device.

Fantom External Backup

The Pictures folders are also backed-up to the Fantom external hard drive.

Appdata Backup

The configuration files for the Docker containers are stored to a system folder named appdata. If the container needed to be rebuilt, due to error or hardware failure, a backup of these files could be restored recreating the container.

In addition to the containers this plugin also backs up the flash drive. Each server has this plugin installed and saves backups to the folder: /mnt/user/Backups/Unraid/. The contents of this folder are synchronized by the Syncthing container. A backup is run weekly on each server.
