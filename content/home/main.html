Unraid File Server / NAS

The Unraid file server commonly referred to as the "NAS" (Network Attached Storage) is where all content used by the Kodi boxes to play movies and TV shows on the TVs. In addition to the content for the TVs the server also hosts shares for family members.

There are currently two file servers in use and are off-the-shelf HPE ProLiant MicroServer Gen10 Plus v2 - both were installed in 2024.

One server is in High Point and named HAVEN-NAS3, it serves as the primary server. The other server is in Adam's apartment in Charlotte and named HAVEN-NAS4. Currently, the servers are near-mirrors of each other, kept in sync with software.

The operating system is a Linux-based OS named Unraid by LimeTech.
