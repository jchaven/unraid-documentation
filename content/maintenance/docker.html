Updating Docker Containers

Updating containers is easily done on the Docker tab on the Unraid management interface. Rarely is there an issue but, if there is one it is usually resolved by removing the container and re-adding it.

Best practice is to update containers by pressing the button Check for Updates followed by Update All at the bottom of the page. If a container is displaying Not available beside it you can resolve this with the button Check for Updates.

Avoid updating containers if they are actively doing something. e.g. SAB is downloading a file or Sonarr is processing files.

After the containers have been updated the respective application (SAB, Sonarr, etc.) will be automatically restarted. The Unraid server itself should not require a restart.
